## Question Answer 
    - 1️⃣ What is the difference between var, let, and const?
    - Ans: 
          -var can be declared repeatedly
            let and const cannot be declared repeatedly
            let can be changed but const cannot be changed.
            var is Function Scoped, let & const is Block Scoped 

    -2️⃣ What is the spread operator (...)?
    -Ans: 
        -The Spread Operator is three dots (...). 
        Its function is to separate the elements from 
        an array or object and place them somewhere else and spread them.

    -3️⃣ What is the difference between map(), filter(), and forEach()?
    -Ans: 
        -map(), filter(), and forEach() work with arrays.
        -forEach() works like a for loop. It performs an operation on each element of 
         the array but does not create a new array itself. It
         only needs to perform some action by looping (such as displaying data or saving it to a database).
        -map() performs an operation on each element of the array and creates a new
         array based on the results.
         Does not change the original array.
        -filter() tests against a condition. Creates a new array of elements that meet
         the condition.
